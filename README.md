This is my Python program modeling a random movie recommendation system — managing genre selection, film generation, and user interactions while utilizing fundamental programming principles.

This project leverages Python dictionaries, functions, the random module, and control flow to define and manage different entities, such as movie genres, film catalogs, and a session-based recommendation history.

I have also included list manipulation, string manipulation, conditional logic, loops, randomization, and state management, illustrating my proficiency in these Python concepts.
